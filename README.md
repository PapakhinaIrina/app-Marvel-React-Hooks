> # **Marvel** information portal :superhero_man:

>>React приложение на хуках

## Разработчик [Ирина Папахина](https://github.com/PapakhinaIrina) 
___
### Стэк: React Hooks, React Router, Yup, Formik, JS, CSS.

___
